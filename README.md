# Cortex-Chatbot


# 📝 Incident Intelligence Q&A Chatbot

An interactive Streamlit web application powered by **Snowflake Cortex Analyst**. 
---

## 🚀 Features
- **Natural Language to SQL**: Converts plain English queries into real-time analytical data.
- **Predefined Examples**: Sidebar triggers for quick navigation and testing of common metrics.
- **Interactive Data Display**: Rich data tables that can be easily scanned and filtered inside the UI.
- **SQL Transparency**: Expandable drawer showing the exact SQL statement generated by the AI model.
- **Data Export**: One-click downloading of query results into CSV spreadsheets.

---

## 🛠️ Prerequisites
- Python 3.8 to 3.11 installed
- Azure OAuth client credentials

---


## 🏃 Running the Application

Launch the server locally by running the following command:
```bash
streamlit run cortex_chatbot.py
```

The application will spin up a local web server and automatically open in your default browser at `http://localhost:8501`.
